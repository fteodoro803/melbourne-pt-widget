# Tram Widget
This application makes gives users widgets for easy access to information on PTV. 

### Dependencies
- python 3.10
- requests 2.31.0
- pytest 8.1.1

### Setup
For the program to work, attach your User/Developer ID and API Key in the *config.ini* file.
